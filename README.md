# WhatRU - แอปพลิเคชันตรวจสอบความปลอดภัยของไฟล์

**WhatRU** เป็นแอปพลิเคชันบนสมาร์ทโฟนที่ถูกพัฒนาขึ้นเพื่อช่วยให้ผู้ใช้งานสามารถตรวจสอบความปลอดภัยของไฟล์ดิจิทัลก่อนนำไปใช้งาน โดยนำหลักการทางนิติวิทยาศาสตร์ดิจิทัล (Digital Forensics) มาประยุกต์ใช้ในรูปแบบที่เข้าใจง่าย แอปพลิเคชันนี้จะช่วยประเมินความเสี่ยงและตรวจจับภัยคุกคามทางไซเบอร์ที่อาจซ่อนมากับไฟล์ได้อย่างรวดเร็ว 

โปรเจกต์นี้เป็นส่วนหนึ่งของรายวิชา ITDS283 Mobile Application Development คณะเทคโนโลยีสารสนเทศและการสื่อสาร มหาวิทยาลัยมหิดล

## ✨ คุณสมบัติเด่น (Key Features)

* **Multi-Engine Scan:** ตรวจสอบไฟล์พร้อมกันจากหลาย Security Engines (เช่น VirusTotal) เพื่อความแม่นยำที่สูงกว่าการใช้ Antivirus ทั่วไปเพียงตัวเดียว
* **On-Demand Scanning:** ระบบทำงานเมื่อผู้ใช้งานสั่งการเท่านั้น ไม่มีการรันอยู่เบื้องหลัง (Background Process) ตลอดเวลา ช่วยประหยัดทรัพยากรเครื่องและแบตเตอรี่
* **Risk Score Dashboard:** นำเสนอผลการสแกนด้วยคะแนนความเสี่ยง (Safety Score) และจัดกลุ่มตามสีที่เข้าใจง่าย (เช่น Clean, Threat)
* **File Integrity Monitoring (StrongBox):** พื้นที่สำหรับจัดเก็บและเฝ้าระวังไฟล์สำคัญ โดยสามารถตรวจจับความเปลี่ยนแปลงโครงสร้างภายในเนื้อหาของไฟล์ได้แม้เพียงเล็กน้อย
* **Scan History:** มีระบบจัดเก็บประวัติผลการสแกนย้อนหลัง เพื่อให้ผู้ใช้ติดตามและประเมินผลไฟล์เก่าได้อย่างสะดวก
* **Deep File Verification:** ตรวจสอบความถูกต้องของนามสกุลไฟล์ (Extension Verification) และการตรวจหาสคริปต์หรือข้อมูลที่ซ่อนอยู่ (Polyglot Detection)

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)

* **API Integration:** VirusTotal API (สำหรับดึงข้อมูลการสแกนจาก Security Engines ชั้นนำ)
* **Backend Services:** Google Firebase (รองรับการทำงานแบบ Real-time และระบบบัญชีผู้ใช้)
* **Security & Cryptography:** Dart.dev Crypto Package (สำหรับทำ Cryptographic Hash Functions เพื่อตรวจสอบความสมบูรณ์ของไฟล์)

## 📱 โครงสร้างและหน้าจอการใช้งานหลัก (Main App Structure)

* **Dashboard Page:** หน้าหลักที่แสดงภาพรวมสถานะความปลอดภัยของอุปกรณ์ สถิติการสแกน และรายการไฟล์ล่าสุด
* **Scan Page:** ส่วนสำหรับเลือกและอัปโหลดไฟล์ มีตัวเลือกสำหรับการเปิด Deep Scan และการใส่ VirusTotal API Key ส่วนตัว
* **Detail Page:** หน้าแสดงผลวิเคราะห์ไฟล์แบบเจาะลึก พร้อมแจ้งเตือนหากพบความเสี่ยง
* **StrongBox Page:** คลังเก็บไฟล์ส่วนตัวที่ได้รับการยืนยันแล้วว่าปลอดภัย
* **News Page:** หน้าต่างรวบรวมข่าวสารอัปเดตเกี่ยวกับความปลอดภัยทางไซเบอร์
